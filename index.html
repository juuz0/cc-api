Codechef API by Nikhil ( juuz0 )

Send a HTTP-request to https://codec-api.herokuapp.com/{method}. Each method requires -
  • handle - for account name
  • count [optional] - to limit the results (not available in user.info)

and returns a JSON-object containing -
  • status - OK/FAILED
  • res (array) - the required result in JSON-format enclosed in an array

user.info

Basic information about the user - rating, username, country, state, city, institution, profession

Usage(Example) - https://codec-api.herokuapp.com/user.info?handle=account_name ( https://codec-api.herokuapp.com/user.info?handle=gennady.korotkevich )

user.subs

Submissions by the user

Usage(Example) - https://codec-api.herokuapp.com/user.subs?handle=account_name&count=number ( https://codec-api.herokuapp.com/user.subs?handle=gennady.korotkevich&count=25 )
